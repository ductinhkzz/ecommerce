## Required

- [Nodejs](https://nodejs.org/en)
- [Postgresql](https://www.postgresql.org/download/)

## Installation

```bash
npm install -g nest yarn
```

```bash
yarn install
```

## Go to .env file

Change your config
```
DB_USERNAME=postgres
DB_PASSWORD=yourpass
DB_DATABASE=test
```

## Running the app

```bash
yarn run start:dev
```
